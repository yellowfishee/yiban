# 首页全面重构设计文档

> 日期：2026-04-12
> 状态：已批准
> 范围：`packages/app/src/pages/home/` + 相关组件

## 背景

当前首页（已打卡态）存在系统性 UI 问题：
- 6 张翻牌轮播 + 长按蓄力交互——不发现、不直觉、不友好
- 垂直堆叠过长（HexagramCard → InspirationDisplay → MeihuaDisplay → 轮播 → 分享 → 提示），手机一屏看不完
- 翻牌状态全局共享，切换卡片丢失翻开状态
- 蓄力进度条 4px 几乎不可见，长按 1.15s 门栏高
- 神兽图标"字在圆里"无辨识度
- 分享按钮紫色渐变与红金暖色体系不一致
- Emoji 当图标跨平台不一致，破坏国风调性

## 设计决策

| 决策点 | 结论 |
|--------|------|
| 重构范围 | 首页全面重构 |
| 翻牌机制 | 合并为单次开签 |
| 信息密度 | 平铺 + 标签页切换 |
| 神兽视觉 | AI 生成插图（初始阶段用占位符） |

## 设计方案：沉浸开签 + 标签卡组

### 1. 首页三大状态

#### 1.1 未打卡态（初始态）

简洁引导页，一屏完成打卡：

- 顶部：应用名"易伴"（小字，不抢视觉）
- 中央：旋转太极 + "梅花易数起卦"副标题 + 一句诗意文案
- 底部：全宽主按钮"梅花起卦 · 打卡领养"
- 去掉当前白色 welcome 卡片框，太极直接浮在背景上

#### 1.2 打卡过渡态（开签动画）

用户点击"梅花起卦"后触发全屏开签动画，约 2.5 秒：

| 阶段 | 时长 | 视觉效果 | 技术实现 |
|------|------|----------|----------|
| 签筒出现 | 0-300ms | 签筒图标从下方弹入 + 弹性 | `translateY` + `cubic-bezier` |
| 签筒摇动 | 300-1000ms | 签筒左右摇 3 次 + 粒子飘出 | `rotate` + 伪元素粒子 |
| 签飞出 | 1000-1800ms | 签从签筒飞出，旋转上升 | `translateY` + `rotate` 组合 |
| 签展开 | 1800-2500ms | 签展开为卡片，渐显内容 | `scaleX` 展开 + `opacity` 渐显 |
| 过渡 | 2500ms 后 | 签/签筒淡出，页面内容淡入 | `opacity` + `visibility` |

技术约束：
- 纯 CSS animation + 少量 JS 状态控制
- 不用 canvas/WebGL（小程序兼容性差）
- 不用 lottie（增加包体积）
- 签筒图标用 SVG 内联实现
- 可跳过：点击任意区域直接跳到最终态
- 降级：`prefers-reduced-motion` 用户直接跳过动画

#### 1.3 已打卡态（主态）

上下两区结构：

```
┌──────────────────────────┐
│  沉浸区（约 45vh）        │
│  ┌──────────────────────┐ │
│  │ [神兽插图]            │ │  ← AI 生成的神兽形象
│  │  卦名 · 卦性质        │ │  ← 如"乾 · 刚健"
│  │  梅花起卦：上X下Y动Z   │ │  ← 精简的梅花信息
│  │                       │ │
│  │  "灵感絮语文本..."     │ │  ← 今日灵感，2-3 行
│  └──────────────────────┘ │
├──────────────────────────┤
│  标签栏                   │
│  [适合|建议|陪同|工作|情感|财运] │ ← 横向标签，可滑动
├──────────────────────────┤
│  内容区（自适应高度）      │
│  ┌──────────────────────┐ │
│  │ 当前选中场景的        │ │  ← AI 生成内容
│  │ 完整文本              │ │
│  └──────────────────────┘ │
├──────────────────────────┤
│  [分享给好友]  [免责声明▼] │  ← 底部操作区
└──────────────────────────┘
```

关键约束：
- 沉浸区固定 45vh，确保标签栏在任何手机上都能露出
- 梅花信息压缩为一行文字（如"上乾下坤 · 动爻三"），不再用独立组件
- 免责声明默认折叠，点击展开

### 2. 沉浸区视觉设计

**背景**：页面整体保持 `#F5F0E8` → `#FDFCF9` 渐变。沉浸区不额外加卡片框，神兽插图直接浮在背景上，配合微弱的墨迹纹理装饰（CSS 实现）。

**神兽插图**：
- 尺寸约 200×200px（手机端），居中
- 用 AI 生成 12 只神兽的统一风格插图（国风水墨 + 神兽形象）
- 初始阶段先用占位符（风格化的文字 + 装饰边框），插图后续替换
- 神兽周围加一层微弱光晕（`box-shadow` 渐变），营造灵性感

**卦名信息**：
- 卦名大字居中（36px），下方小字"· 性质"（20px）
- 颜色：卦名 `#C73E3A`，性质 `rgba(26,43,60,0.6)`
- 字体保持 STSong 衬线体

**梅花信息**：
- 一行小字居中，如 `上乾下坤 · 动爻三`
- 字号 18px，颜色 `rgba(26,43,60,0.4)`
- 不再使用 MeihuaDisplay 独立组件

**灵感絮语**：
- 神兽下方，最多 3 行，超出截断
- 字号 22px，行高 1.8，颜色 `#1A2B3C`
- 保留 typewriter 逐字效果，加速到 30ms/字 + 点击跳过
- 外层不加卡片框，一条细分割线与标签区分隔

### 3. 标签页 + 场景内容区

#### 3.1 标签栏

```
┌──────────────────────────────────┐
│  今日适合 | 处事建议 | 情绪陪同 | 工作发展 | 情感沟通 | 财运参考  │
│  ━━━━━━                          │  ← 选中态下划线
└──────────────────────────────────┘
```

视觉规格：
- 标签文字 24px，未选中 `rgba(26,43,60,0.4)`，选中 `#C73E3A`
- 选中态下方 4px 红色下划线（`#C73E3A`），宽度与文字等宽
- 标签间距 32px，整体水平 padding 24px
- 横向可滑动（Taro `ScrollView`），不显示滚动条
- 首个标签默认选中

交互：
- 点击标签切换，内容即时显示
- 内容未生成时显示骨架屏
- 无翻开交互，内容直接可见

#### 3.2 场景内容区

视觉规格：
- 无卡片框，内容直接渲染在页面背景上
- 左上角小字标签（如"今日适合"），颜色 `#D4AF37`，字号 20px
- 下方一条金色细线分隔（`1px solid rgba(212,175,55,0.3)`）
- 正文 24px，行高 1.8，颜色 `#1A2B3C`
- 最小高度 200px，最大不限（自然撑开）
- 底部间距 24px

加载态：3 行骨架条（宽度 80%/60%/70%），金色渐变闪烁，不用 spinner。

空态：显示"神兽暂时沉默，请稍后再试" + 重试按钮。

#### 3.3 缓存策略

- 切换标签时，已加载内容保留在内存
- 首次进入某标签触发 API 请求（如果本地无缓存）
- 复用现有 `agentContents` 状态，不改变数据层

### 4. 底部操作区

```
┌──────────────────────────────────┐
│  [ 分享给好友 ]    免责声明 ▼     │
└──────────────────────────────────┘
```

**分享按钮**：
- 全宽按钮，圆角 16px，高度 72px
- 背景：`linear-gradient(135deg, #C73E3A 0%, #E64843 100%)`——统一到红金体系
- 文字白色 28px
- 去掉紫色渐变

**免责声明**：
- 默认折叠，显示"免责声明 ▼"（20px，`rgba(26,43,60,0.35)`）
- 点击展开完整声明文本
- 不再使用独立 Disclaimer 组件，内联在底部操作区
- 去掉 emoji 图标

### 5. 色彩体系统一

整个首页只用一套色彩 token：

| Token | 值 | 用途 |
|-------|-----|------|
| `--color-primary` | `#C73E3A` | 主色（按钮、选中态、卦名） |
| `--color-gold` | `#D4AF37` | 金色装饰（分割线、标签、边框） |
| `--color-text` | `#1A2B3C` | 正文 |
| `--color-text-secondary` | `rgba(26,43,60,0.6)` | 辅助文字 |
| `--color-text-hint` | `rgba(26,43,60,0.35)` | 提示文字 |
| `--color-bg` | `#F5F0E8` | 页面背景 |
| `--color-bg-light` | `#FDFCF9` | 浅背景 |

不再出现紫色、蓝色等跳出色，所有色彩在红金暖色体系内。

### 6. 组件清理清单

| 组件 | 动作 | 理由 |
|------|------|------|
| `HexagramCard` | 重构为沉浸区子组件 | 去掉浮动动画、云 emoji、超大 padding |
| `MeihuaDisplay` | 废弃 | 信息压缩为一行文字 |
| `MysticParticles` | 废弃 | 仅用于翻牌场景，翻牌机制已去除 |
| `RevealText` | 废弃 | 仅用于翻牌正面，标签页直接展示 |
| `MoodSelector` | 暂保留但不展示 | 后续接入心情选择再启用 |
| `InspirationText` | 优化 | 加速 typewriter + 增加跳过 |
| `ShareButton` | 重构 | 统一色彩，紫色→红色 |
| `Disclaimer` | 废弃独立组件 | 内联到底部操作区 |
| `AgentContentList` | 废弃 | 未使用的遗留组件 |
| `carousel-flip` 样式 | 全部删除 | 翻牌机制已去除 |

保留的组件：`HexagramSymbol`（后续替换插图）、`InspirationDisplay`（简化）
